# logia

This repository contains solutions of tasks presented at previous editions of a Polish coding competition ["Logia"](https://logia.oeiizk.waw.pl/home).

## Repository structure

Naming convention: `L[YY]_E[X]_[Z]_[task_name].py`

- `[YY]` – edition number
- `[X]` – number of the competition stage
- `[Z]` – task number (if applicable)

Directories:
- `notebooks` – notebooks created for purposes of explaining some programming concepts
- `prep` – solutions implemented by me to get familiar with the competition
- `solutions` – code pair-programmed with a student

Each `.py` file contains a link to a task overview from a Logia archive.
